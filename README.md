Tic Tac Toe (Java Swing)
This project is a simple two-player Tic Tac Toe game implemented using Java Swing. It provides a graphical interface where two players can take turns marking X and O on a 3x3 grid. The game detects wins, draws, and includes a reset option.

- Features:
1. Graphical user interface built with Java Swing
2. Turn-based gameplay (X starts first)
3. Win detection for all valid combinations
4. Draw detection when the board is full
5. Reset button to restart the game
6. Beginner-friendly and easy-to-understand code

- Requirements:
Java Development Kit (JDK) 8 or higher

- How to Compile and Run:
1. Open a terminal or command prompt inside the project directory.
2. Compile the Java file:
javac src/TicTacToe.java
3. Run the program:
java src.TicTacToe

- How to Use:
1. Player X begins the game.
2. Players alternate turns by clicking on the grid buttons.
3. The game displays the winner when three matching symbols align horizontally, vertically, or diagonally.
4. If all cells are filled without a winning combination, the game ends in a draw.
5. The Reset button clears the board and starts a new game.
